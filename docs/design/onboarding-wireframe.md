# CotePT 온보딩 화면 정의서

## 📱 전체 플로우 개요

```
멘티 플로우: profile-setup → baekjoon-verify → skill-analysis → complete
멘토 플로우: profile-setup → baekjoon-verify → skill-analysis → mentor-tags → mentor-intro → complete
```

---

## 🎨 Step 1: 프로필 설정 (profile-setup)

### 화면 구성
```
┌─────────────────────────────────────────┐
│                                         │
│          코테피티에 오신 걸            │
│            환영합니다! 🎉              │
│                                         │
│      프로필을 설정해주세요             │
│                                         │
│         ┌─────────────┐                │
│         │             │                │
│         │   [📷]      │  ← 프로필 이미지 │
│         │             │    (클릭 업로드) │
│         └─────────────┘                │
│                                         │
│      닉네임 *                          │
│      ┌───────────────────────────┐    │
│      │                           │    │
│      └───────────────────────────┘    │
│      2-20자, 한글/영문/숫자           │
│                                         │
│                                         │
│              [다음 단계 →]             │
│                                         │
└─────────────────────────────────────────┘
```

### 유효성 검사
- 닉네임: 필수, 2-20자
- 프로필 이미지: 선택, 5MB 이하

### 동작
- 다음 버튼: `useCreateBasicProfile` mutation 호출
- 성공 시: baekjoon-verify 단계로 이동

## 📱 반응형 디자인

- Desktop: 중앙 정렬, 최대 600px
- Mobile: 전체 너비, 패딩 16px

## 🎨 컬러 시스템

- Primary: 온보딩 진행 버튼
- Secondary: 이전 버튼
- Success: 완료 상태
- Warning: 인증 대기
- Error: 유효성 검사 실패

## ⌨️ 키보드 단축키

- Enter: 다음 단계
- ESC: 모달 닫기
- Tab: 포커스 이동
