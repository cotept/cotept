# CotePT UI/UX 기획서

## 🎯 서비스 개요

**CotePT (코테피티)**는 코딩테스트 준비생을 위한 **1:1 실시간 멘토링 플랫폼**입니다.

### 핵심 기능
- 🎥 **실시간 멘토링**: WebRTC 음성 + 코드 공유 에디터
- 📹 **세션 녹화**: VOD 다시보기 서비스
- 🏆 **멘토 검증**: 백준 Platinum 3+ 자격 확인
- 🔐 **간편 인증**: 소셜 로그인 + PASS 본인인증
- 💎 **구독 서비스**: 프리미엄 VOD 콘텐츠
